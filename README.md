# 安装说明参考
https://www.bilibili.com/video/BV1xA4m1P7qH
1. 安装nodejs
2. 安装postgre数据库
3. 解压项目，cd到项目目录，执行npm -ci安装项目依赖
4. 编辑config/config_example.js里面的内容，根据提示修改数据库等信息，并修改文件名为config.js
5. 在项目目录里，执行node serve.js启动项目

你可以使用pm2或者tmux做进程守护避免终端退出以后程序中止
或者你可以使用linux面板自带的nodejs把项目目录添加进去，然后利用面板来启动
